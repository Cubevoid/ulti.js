Ulti.js Documentation

Getting Started

First you must link the library and stylesheet in the <head> element. 
<script src="ulti.js"></script>
<link rel="stylesheet" type="text/css" href="ulti.css" />

API Methods

constructor(width, length, orientation = "vertical", enableDragging = true, debug = false)

To create a blank field, use the constructor of the UltiBoard object: 
const ulti = new UltiBoard()
The constructor takes the following arguments:
  1. width (in pixels)
  2. length (in pixels)
  3. orientation ("vertical" or "horizontal")
  4. enableDragging (boolean)
  5. debug (boolean)
Note: the following methods are all performed on the UltiBoard object.

At any point, you can set the debug flag to enable or disable log messages in the browser console.

ulti.debug = true

addPlayer(x, y, name, jerseyNumber, type = "offense", rotation = 0) => newPlayer

We can create a new player using the addPlayer method: 
ulti.addPlayer(30, "50%", "Marques Brownlee", 13, "defense", 90)
The addPlayer method takes the following arguments:
  1. x (in pixels or as a percentage in the form of a string)
  2. y (in pixels or as a percentage in the form of a string)
  3. name (the player's name/label)
  4. jerseyNumber (the player's jersey number)
  5. type ("offense" or "defense")
  6. rotation (in degrees)
The method returns the object representing the new player.

filterPlayers(filter)

Filters the players array based on a given filter callback. 
ulti.filterPlayers((player) => {
  return player.x !== 0
})

updatePlayers()

Updates the players in the DOM from the UltiBoard's players array.
Use this after manually modifying the players array. 
ulti.players[0].name = "Colin De Vlieghere"
ulti.players[0].number = 23
ulti.updatePlayers()

showDisc()

Shows the disc on the field.

hideDisc()

Hides the disc.

setDiscOffset(x, y)

Moves the disc to the specified coordinates in pixels (from the top left of the field, or from the disc's owner if applicable). 
ulti.setDiscOffset(50, 200)

setDiscOwner(newOwner)

Sets the new owner of the disc, i.e. the player holding the disc. 
ulti.setDiscOwner(ulti.players[0])
The players parameter should be a Player object.

showNames()

Force shows all player names and numbers in the form of tooltips.

hideNames()

Hide all player names and numbers tooltips. This is the default.

clear()

Removes all players and the disc from the field.

clearDrawings()

Removes all arrows/drawings from the field.

vertStack(defenders = false)

Sets the field to the Vertical Stack preset. To show the defense players, set defenders to true.

hoStack(defenders = false)

Sets the field to the Horizontal Stack preset. To show the defense players, set defenders to true.
